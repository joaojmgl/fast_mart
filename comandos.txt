TODOS OS COMANDOS DEVEM SER RODADOS DENTRO DA PASTA 'NODE_SEQUELIZE':

-> npm install
Usado para instalar as dependências

-> npm run dev
Usado para rodar o servidor

-> sequelize migration:create --name=add_endereço_in_company
Usado para criar um novo arquivo de migração no Sequelize CLI ("create-users" seria o nome)

-> npx sequelize db:migrate
Usado para aplicar migrações ao banco de dados usando o Sequelize CLI (rodar as migrations)

-> npx sequelize-cli db:migrate:undo
Usado para reverter migrações

!! lembrar de criar o banco de dados "trabalho" ou mudarem no código para ter outro nome, e também mudar a senha do banco de dados de vocês.

-> streamlit run analise.py 
Usado para executar o dashboard de analise de dados. (Deve ser executado dentro de ./NODE_SEQUELIZE/src/data-analisys)
